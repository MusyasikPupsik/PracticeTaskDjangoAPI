# Django REST API - Практическая работа

Финальное задание по производственной практике. Данный проект представляет собой REST API, построенный на базе Django и Django REST Framework (DRF) в соответствии с предоставленной структурой базы данных (`models.py`).

## Реализованный функционал
* Настроено автоматическое отображение моделей в СУБД SQLite через миграции.
* Реализованы сериализаторы (`serializers.py`) для валидации данных и преобразования их в JSON.
* Созданы контроллеры (`views.py`) на базе `ModelViewSet` для полной поддержки CRUD-операций.
* Настроена автоматическая маршрутизация (`urls.py`) для взаимодействия с фронтендом.

## Основные эндпоинты API
* `/api/users/` — Управление пользователями
* `/api/projects/` — Управление проектами
* `/api/hypotheses/` — Управление гипотезами
* `/api/teams/` — Управление командами

## Инструкция по локальному запуску

1. Установите зависимости:
```bash
pip install django djangorestframework
```

2. Выполните миграции базы данных:
```bash
python manage.py migrate
```

3. Запустите локальный сервер:
```bash
python manage.py runserver
```

После запуска интерактивный интерфейс API будет доступен по адресу: http://127.0.0
